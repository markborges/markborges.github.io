Current Version

You can download the latest version of the BC Ideas by BC Repositories app below. Please note, this is a free download, and you should not provide any payment information in order to download it.

  • Business Central Repositories_BC Ideas by BC Repositories_24.0.4.0

This extension uses the Object Range 61500-61999. If you have any objects defined in this range, you may run into conflicts and I advise you don’t install this extension.

How to Use the BC Ideas App

When you install the latest version, please search for “BC Idea Feature Management” from the Tell Me search. You should find a list of all BC Ideas included on this version (and their statuses). You can read more details about each one of them, and Enable or Disable them individually, per company.
